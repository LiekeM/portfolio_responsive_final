OPEN_HRTG

Built with JavaScript, HTML and CSS

OPEN HRTG; is a digital documentation database of ancient cultural heritage. By collecting users' 3D-scans of objects in musea, heritage sites and institutes, we create a free acces database to objects and places of historical value through digital technologies. The user can leave their personal details and a description of the object they would like to 3D-scan.

Weather App

Built with JavaScript, HTML and CSS

What Should I wear is a weather app, presenting the forecast in a fresh and playful manner. By displaying emojis of clothing and accesoires instead of weather icons, this app gives the user an indication of what to wear, today, tomorrow or the next five days. What Should I Wear is using Open Weather API to get realiable weather data collection and forecast, which is converted into emoji's that intuitivle give the user an indication of the weather.
